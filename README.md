### 各模块详解

#### 电源模块
1. 选材：12V锂电池，DC电源适配器，MP2338GTL-Z
2. 电路设计
   
   使用带DC接线头的12V锂电池组作为电源，使用DC电源适配器将电源接入电路板中
   
   使用**MP2338GTL-Z**DC-DC电源芯片降压至5V
   
   再使用**AMS1117-3.3**LDO将5V转化为3.3V供PCB使用
